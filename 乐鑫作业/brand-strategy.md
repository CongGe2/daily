# 潜江虾局 · 品牌形象与包装设计策略

## 一、品牌定位

| 维度 | 定义 |
|------|------|
| 品牌名 | 潜江虾局 QIANJIANG SHRIMP BUREAU |
| 定位 | 年轻人的夜宵社交预制菜品牌 |
| 口号 | 开锅即虾局 / Get the Party Crawling |
| 目标消费者 | 18-28岁大学生与城市青年，宿舍/聚会/露营/外卖场景 |
| 品牌人格 | 爽快、有梗、懂吃、不装 |
| 核心差异 | 把"潜江产地"做成品牌信任状，把"虾局"做成社交场景IP |

## 二、视觉系统

### 色彩方案

| 色名 | 色值 | 用途 |
|------|------|------|
| 虾红 | #E8462C | 主色·品牌识别 |
| 料金 | #F5A623 | 辅色·蒜蓉/品质感 |
| 局黑 | #1C1915 | 文字·力量感 |
| 壳白 | #FFF8F0 | 底色·包装底 |
| 麻棕 | #8B3A1F | 十三香口味色 |

### 口味色彩系统

| 口味 | 主色 | 色值 |
|------|------|------|
| 麻辣 | 烈焰红 | #E8462C |
| 蒜蓉 | 金蒜黄 | #F5A623 |
| 十三香 | 香料棕 | #8B3A1F |

### 字体规范
- 品牌标题：自定字标（手写感+力量感）
- 中文正文：阿里巴巴普惠体 / 思源黑体
- 英文辅助：Inter Bold

### 超级符号
- "虾局"二字合体字标——虾尾勾连"局"字的丿画
- 品牌IP：潮酷小龙虾"虾局长"——墨镜+大钳+金链

## 三、包装系统

### A. 口味包装盒（麻辣/蒜蓉/十三香）
- 结构：天地盖纸盒 + 开窗
- 材质：食品级白卡纸 + 内衬铝箔保温
- 视觉：口味色大面积铺底 + 虾局长IP居中 + 口味标签
- 背面：加热说明三步图 + 营养成分表 + 产地溯源码

### B. 冷链外箱
- 材质：瓦楞纸 + 防水涂层
- 视觉：虾红品牌色 + 白色品牌字标 + "冷藏运输"标识

### C. 调料包
- 三连包撕线设计
- 口味色区分 + 大字口味名

### D. 外卖袋
- 牛皮纸+品牌色带
- 虾局长IP大图 + "开锅即虾局"口号

### E. 加热说明卡
- 三折小卡
- 正面：虾局长教你三步加热
- 背面：潜江龙虾产地故事

## 四、传播场景

- 电商详情页（淘宝/京东/抖音电商）
- 社媒海报（小红书/朋友圈）

- 打卡墙/快闪店（"潜江虾局·今夜开虾"）
