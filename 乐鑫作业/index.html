核心发现：小龙虾预制菜的竞争不在"谁的小龙虾更好吃"，而在"谁的包装让年轻人更想拍照发朋友圈"。口味差异化的终点不是味觉，是视觉——货架上三秒识别、社交平台一秒种草。

超级符号 · 虾局长 🦞

潮酷小龙虾人格化IP——墨镜、大钳、金链。不是可爱的卡通虾，是有态度、懂吃、能组局的夜宵文化代言人。

Packaging System

全套包装展示

从口味包装盒到冷链外箱、外卖手提袋——完整的五件套包装链路，每一件都独立成立，合在一起形成统一品牌识别。

🌶️ 麻辣口味

  • 色彩：烈焰红全面铺底
  • 记忆点：辣椒图标 + "开锅即辣"
  • 结构：天地盖 + 开窗 + 铝箔内衬

🧄 蒜蓉口味

  • 色彩：金蒜黄铺底，温暖食欲感
  • 记忆点：蒜瓣超级图形 + "蒜你识货"
  • 差异化：三款中最具大众接受度

🫚 十三香口味

  • 色彩：香料棕，沉稳高级
  • 记忆点：十三香图谱纹样 + "一锅十三味"
  • 差异化：送礼属性最强

📦 冷链快递外箱

  • 材质：瓦楞牛皮纸 + 防水涂层
  • 视觉：品牌红 + 白色字标 + 冷藏标识
  • 开箱体验：内侧印"虾局长欢迎您"

🛍️ 外卖手提袋

  • 材质：牛皮纸 + 品牌红色带
  • 视觉：虾局长IP大幅面 + 口号
  • 社交属性：拿在街上就是行走的广告牌
