01 · Problem Discovery

不到 4㎡ 的生活容器

澳门高校宿舍标准间面积约 12㎡，两人共用，人均居住面积不足 4㎡。传统宿舍家具——上床下桌 + 衣柜——无法适应学生"学习-睡眠-社交-储物"多场景快速切换需求。核心矛盾：空间固定，但使用场景高度动态。

🔍 现象

书桌堆满杂物无法学习；换季衣物无处收纳；室友作息冲突时无独立空间；外卖/快递包装占据走廊。

🎯 设计机会

如何在不改造宿舍结构的前提下，用模块化产品系统提升空间使用效率？关键在于"灵活切换"而非"更多柜子"。

📐 约束条件

不能钻孔/不可永久改造 · 需适配不同宿舍户型 · 成本控制在学生可承受范围 · 单人可独立安装。

02 · User Research

12 位学生的真实声音

我主持了 12 位澳门高校住宿生的深度访谈（单人 30-45 分钟），覆盖大一至大四、不同专业、不同宿舍楼型。同时收集了 38 份宿舍实拍照片用于场景分析。

😤 最大痛点

76% 受访者表示"桌面永远不够用"——吃饭、学习、化妆全部在同一张桌上完成，场景切换靠"把东西推到一边"。

📦 收纳行为

超过半数学生使用自购收纳盒（塑料抽屉、挂袋），但产品之间不兼容，空间利用率反而下降。

🏠 改造意愿

83% 愿意购买改善宿舍空间的方案，但心理价位集中在 200-500 元，且要求"毕业能带走或转卖"。

"我不是不想整理——是这房间根本没有'整理'的逻辑。所有东西都堆在一起，收拾完两天又乱了。" —— 大三受访者

👤 典型用户画像 · 小李

大三计算机系，宿舍面积约 5.5㎡（两人间）。桌面同时承载：笔记本电脑 + 外接显示器 + 机械键盘 + 外卖 + 水杯 + 手机充电线。每天至少 3 次"把东西推到一边"的动作。衣柜深度 55cm 但无分层，换季衣物堆在行李箱里。

核心需求：桌面扩展、衣物分区、床边随手收纳

📐 宿舍户型实测数据

测绘了 3 栋宿舍楼共 8 间标准间：桌面 120×60cm（离地 75cm）、床铺 200×90cm（离地 175cm）、衣柜 90×55×180cm、过道宽度 80-110cm。天花板高度 260cm，不可改造。

设计约束：所有产品须通过 80cm 过道搬运 · 单人可安装 · 毕业可拆卸带走

03 · Design Solution

模块化系统：搭积木一样收纳

核心设计理念——不是做"更多家具"，而是做一套"空间乐高"。3 件模块化产品可以独立使用，也可以互相组合，适配不同宿舍户型和使用场景。

🧱 A 模块：桌面扩展板

折叠式扩展台面，展开后桌面深度从 60cm 延伸至 90cm。折叠时占用厚度仅 6cm，可兼作留言板/照片墙。滑轨结构支持单手操作。

📦 B 模块：堆叠储物箱

3 尺寸可选（衣物/文具/杂物），顶部凹槽与底部凸起咬合，堆叠后稳固不滑落。透明视窗设计无需打开即可看到内容物。可横放/竖放。

🪝 C 模块：床边挂架系统

无需打孔的快装挂架，利用床架结构承重。集成手机支架 + USB 充电走线槽 + 水杯位。解决"上床后什么都想带着"的痛点。

04 · CMF & Prototype

材料、色彩与原型验证

3 套 CMF 方案应对不同宿舍风格：原木暖色系适配老式宿舍、灰白极简系适配新建宿舍、莫兰迪彩色系面向女性用户。1:5 白模快速验证结构比例与组合方式。

🎨 CMF 方案 A · 原木暖色

桦木纹贴面 + 米白色 ABS。适配传统宿舍的木色家具，视觉上"融入原有空间"而非"增加一件东西"。

🎨 CMF 方案 B · 灰白极简

浅灰纹理 + 纯白哑光。面向新建宿舍的现代风格，与白色墙面和金属家具协调。

🎨 CMF 方案 C · 莫兰迪色

雾蓝 + 豆沙绿 + 奶油黄。调研发现女性用户对收纳产品外观要求显著更高，推出彩色系列。

🔧 原型验证

Rhino 7 全尺寸建模，3D 打印 1:5 白模。邀请 4 位受访者进行"桌面模拟摆放"测试，验证模块组合直觉性和尺寸适配性。

05 · Mini Program UI

微信小程序：把收纳数字化

产品之外的数字化延伸——我独立完成全部 UI 设计。核心功能：扫码识别物品 → 收纳方案推荐 → 空间可视化预览。

📱 首页 · 我的宿舍

房间平面图视图，拖拽模块到对应位置预览效果。底部 Tab 导航：首页 / 物品 / 方案 / 我的。

📷 物品识别

拍照或从相册上传，AI 识别物品类型（衣物/书籍/电子/杂物），自动推荐存放模块和位置。

💡 收纳方案

预设 6 种常见宿舍户型的模块搭配方案。用户可以按"最大化书桌空间""最大化衣物收纳"等目标筛选。

🛒 购买与管理

模块产品购买链接、订单追踪、已购模块的电子说明书与安装视频。二手转卖信息发布。
